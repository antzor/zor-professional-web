# Open Graph (OG) Images Directory

This directory contains Open Graph images for social media sharing.

## What are OG Images?

Open Graph images are displayed when pages are shared on social media platforms like Facebook, LinkedIn, Twitter, etc. They create rich previews that increase click-through rates.

## Images Needed:

1. **homepage.jpg** (1200x630px)
   - Main ZOR Professional branding image
   - Used for: Homepage and default fallback

2. **products.jpg** (1200x630px)
   - Product showcase image
   - Used for: Products page

3. **blog.jpg** (1200x630px)
   - Generic blog header image
   - Used for: Blog listing page

4. **og-default.jpg** (1200x630px)
   - Default fallback image for pages without specific OG images
   - Should include ZOR Professional logo and tagline

## Image Specifications:

- **Size**: 1200x630px (Facebook/LinkedIn recommended)
- **Aspect Ratio**: 1.91:1
- **Max File Size**: 8MB
- **Format**: JPG or PNG
- **Text**: Should be readable at small sizes
- **Safe Zone**: Keep important content within center 80%

## Best Practices:

1. Include ZOR Professional logo
2. Keep text large and readable
3. Use high-contrast colors
4. Avoid small details that won't be visible in thumbnails
5. Test preview on Facebook Sharing Debugger and LinkedIn Post Inspector

## Testing Tools:

- Facebook: https://developers.facebook.com/tools/debug/
- LinkedIn: https://www.linkedin.com/post-inspector/
- Twitter: https://cards-dev.twitter.com/validator
