## Image Specifications:

- **Aspect Ratio**: 16:9 (recommended)
- **Recommended Size**: 1200x675px
- **Max Size**: 2MB per image
- **Format**: JPG or WebP (preferred)
- **Alt Text**: Defined in blog post content files

## Naming Convention:

Use kebab-case matching the blog post slug for easy identification:
- `sustainable-paper-solutions.jpg` (matches slug: sustainable-paper-solutions)
- `whitelabel-benefits.jpg` (matches slug: whitelabel-benefits)
- `paper-towel-types.jpg` (matches slug: paper-towel-types)
